Описание задачи:
API - https://jsonplaceholder.typicode.com

Шаги
1. Отправьте запрос GET, чтобы получить все сообщения (/posts).
Ожидаемый результат:
Код состояния 200.
Список в теле ответа — json.
Сообщения сортируются по возрастанию (по id).

2. Отправьте запрос GET, чтобы получить пост с id=99 (/posts/99).
Ожидаемый результат:
Код состояния 200.
Информация о сообщении верна: userId — 10, id — 99, заголовок и тело не пусты.

3. Отправьте запрос GET, чтобы получить пост с id=150 (/posts/150).
Ожидаемый результат:
Код состояния 404.
Тело ответа пусто.

4. Отправьте POST-запрос, чтобы создать сообщение с userId=1 и случайным телом и случайным заголовком (/posts).
Ожидаемый результат:
Код состояния 201.
Информация о сообщении верна: заголовок, тело, userId совпадают с данными из запроса, id присутствует в ответе.

5. Отправьте запрос GET, чтобы получить пользователей (/users).
Ожидаемый результат:
